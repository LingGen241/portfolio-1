HRIS for Pay Equity

Led a team to develop a Human Resource Information System that integrated 10+ existing systems, identifying and addressing pay disparities. The project emphasized stakeholder collaboration, workflow design, and building a transparent, scalable solution to promote workplace equity.

We believe every team member deserves fair compensation. By centralizing data, we spotlight inequities and empower action.

Key Features

  • Centralized Compensation Management: Allows stakeholders to retrieve data on employees across departments based on training, performance, and financial information, enabling faster evaluations.
  • Employee Training Modules: Provides resources to enhance employee training, improving evaluations and competitiveness while clearly defining expectations.
  • Integration of Compensation and Payroll: Enhances payroll software to accurately track financial data, ensuring data privacy and user-friendly tracking.
  • Scalability of HRIS: Builds on existing HR systems for practical, scalable employee evaluations, avoiding complete redesigns.
  • Disparity Audits: Integrates software to detect potential biases in employee data, proactively mitigating unfairness risks.
  • Real-time Analytics Dashboard: Interactive dashboards showing compensation trends, performance metrics, and budget analytics for proactive decision-making.
  • Compliance Monitoring System: Automated checks to align compensation decisions with legal and internal policies, reducing risks.
  • Employee Self-Service Portal: Interface for employees to view compensation and performance data, increasing transparency and reducing HR workload.
  • Reporting and Documentation Tools: Automated generation of pay and performance reports for consistent auditing.
  • Integration Capabilities: Connects existing HR and payroll systems, maintaining data consistency and leveraging technology investments.

Solution Wireframe

The wireframe outlines a user interface for managing employee records with the following functionalities:

  1. View recently uploaded records in a mass view.
  2. Filter records by department (e.g., HR, Legal, Payroll).
  3. Filter records by employee name based on selected department.
  4. Filter records by type (e.g., compensation data, training file, evaluation).
  5. Navigate to a new interface to update selected records.
  6. Remove selected data.
